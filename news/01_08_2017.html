BetterDiscord Comeback!

January 8, 2017

If your Discord app client has updated then there is a high chance that it stopped BetterDiscord from loading. The deveoper has fixed the issue of local.Storage and has released a new launcher fixing any current issues with BetterDiscord. There has also been a new seperate storage system for plugin developers to use.

New Features:

  • Plugins developers can now use bdPluginStorage for their plugins!

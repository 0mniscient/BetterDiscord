Installing on Linux

Requirements

  • Node.js
  • Electron

Installation

Info: Options are disabled for testing purposes

Now this process is a bit complicated, you have 2 options on how you want to download the installer.

  • Option 1: Clone/Download Zip the makers entire Github Repo.
  • Option 2: Download the files from the makers Github.

Now for the more complicated part of this installation.

  • Launch terminal and run npm install
  • Now in terminal run ./node_modules/.bin/electron ./src and it will open the installer
  • If the installer freezes during the installation, run sudo chown yourname /usr/share/discord-canary/resources
  • Rename Utils.js to utils.js in the BetterDiscord installation directory
  • After the installation finishes just click "Abort". It doesn't actually abort anything it's just a placeholder.

Updates

BetterDiscord and Discord updates are 2 things to keep in mind.

BetterDiscord Updates

BetterDiscord will update automatically in a Discord restart (CTRL+R) you can check in the news page for updates or on the BetterDiscord server

.

Discord Updates

Every time there is a big Discord update BetterDiscord will get removed from Discord. How do you fix this? Just reinstall BetterDiscord and you will not loose anything!

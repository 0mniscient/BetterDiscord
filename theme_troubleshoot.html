FAQ

Here are answers to frequently asked questions about themes.

Empty Theme List

Please follow these steps to get your themes showing up again!

  • First make sure you saved the theme to 
    Windows: %appdata%\betterdiscord\themes\
    Mac: ~/Library/Preferences/BetterDiscord/themes
  • Make sure the themes file extension is .theme.css
  • Make sure the themes 1st line of CSS contains the META tag
  • Now restart your Discord app
  • If step 4 didn't work please reinstall BetterDiscord

Ghost Theme

A Ghost Theme is when when you removed a theme from your theme folder and restarted Discord and it still shows up under "Themes"

Important: This will erase all your themes.
  • First lets start clean and restart your Discord
  • Now open the inspect element tool inside Discord by doing CTRL+SHIFT+I
  • Click the "Console" tab
  • Enter bdthemes = {};

Remove CSS manually

You might want to remove some CSS manually because you can't access your Discord settings or you just really mest up Discord but don't worry we can still save your Discord.

  • Open the inspect element tool inside Discord by doing CTRL+SHIFT+I
  • Click the "Console" tab
  • Enter $('#customcss').empty()
  • Go back to the Custom CSS tab remove the CSS and uncheck "Live Update" and "Autosave" and click "Save"
  • Now just restart Discord

Change background image

Now you might be new to CSS and have a transparent theme already installed on Discord and want to change the background image well it's really simple.

  • Open the theme file with a text editor (Recommended: Atom,Notepad++)
  • Search for .app and .callout-backdrop in the CSS. In some cases they might be in the same line of CSS
  • Replace the link(s) with your image link. Please note that the link needs to be HTTPS:// and end with a file extension such as png,jpg,gif etc
  • Now restart Discord to see your new beautiful (hopefully) background!

Theme Hosting

Many people ask where is the best place to host your Theme to make it easy for users to download and update.

  • Github - Most recommended site to host your themes/plugins
  • Google Drive - Ok site to host themes/plugins
  • PasteBin/Ghostbin - Not recommended at all to host themes/plugins

Meta Tag

A Meta tag is the first line of CSS/Javascript that is required for every Theme/Plugin to have so BetterDiscord can identify the file.

  • Example of a Theme's Meta Tag 
    //META{"name":"First_Theme","description":"My First Theme.","author":"Omniscient","version":"1.00"}*//

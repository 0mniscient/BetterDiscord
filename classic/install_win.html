BetterDiscord

Installing on Windows

Important: By installing BetterDiscord you agree to not contact Discord Developers about BetterDiscord issues and will instead ask for support in the BetterDiscord Server.

It only takes a few minutes to set up BetterDiscord. If you encounter a problem and can’t find the solution here, please submit a GitHub issue and we will try to solve it.

Installation

First step head on over to the BetterDiscord website

  • Click the big download button for Windows.

  • Extract the zip file and you should and you should see a total of 5 files.

  • Double click the file called BetterDiscordWI.exe then hit "run" and do the following. If you encounter any errors throught this process please try reinstalling Discord then BetterDiscord before sumbitting an issue or contacting us.

  • Now you have installed BetterDiscord congratulations! Now you can start downloading Themes and Plugins.

    Updates

    BetterDiscord and Discord updates are 2 things to keep in mind.

    BetterDiscord Updates

    BetterDiscord will update automatically in a Discord restart (CTRL+R) you can check in the news page for updates or on the BetterDiscord server

    .

    Discord Updates

    Every time there is a big Discord update BetterDiscord will get removed from Discord. How do you fix this? Just reinstall BetterDiscord and you will not loose anything!
